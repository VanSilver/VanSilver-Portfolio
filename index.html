Services

Embedded & IoT Development

I develop embedded and IoT systems on microcontroller platforms, performing specific tasks such as data acquisition, sensor interfacing, and communication with cloud services using protocols like MQTT and HTTP.

Hardware PCB Design

I design PCBs to perform the specific tasks my embedded system performs. This includes selecting components, creating schematics, and laying out the PCB to ensure proper functionality and reliability.

Application Development

I write applications on both windows and mobile platforms to deploy to embedded and IoT systems that I design, and also write tools to do some automated work.

Human Dectect Radar Sensor

A human presence detection system based on the HLK-LD2402radar module. The project integrates Doppler radar sensing for accurate motion and presence detection, offering higher stability and penetration compared to traditional PIR sensors. It communicates with the main controller via UART for smart lighting and security applications.
